How to use this program:
-To add a free node, type the node name
into the top text box and click "Add new node"
while no node is selected

-To make a connection between two existing
nodes, left-click one node and then right-click
another

-To make a connection from an existing node
to a new node, left-click a node, type the
name of the new node into the top text-box,
and then click "Add new node"

-To save your work, select "Export Network".
This will save the network to your as a text file.
To reload the network, paste it into the bottom
text-box and click "Import Network".
To start, either click Start with the text field blank,
or paste in the URL of an image you want as your background
